## 📋 Descripción del Proyecto
Este proyecto es un portafolio profesional desarrollado para demostrar los fundamentos de la web moderna, cumpliendo con los estándares de estructura semántica **HTML5**, diseño responsivo con **CSS3** y lógica de persistencia de datos en el cliente mediante **JavaScript** y **LocalStorage**.

*   **JavaScript (Vanilla):** Lógica de autenticación local y manejo de estados.

## 🧠 LocalStorage vs SessionStorage

> [!NOTE]
> La diferencia principal es en cuanto al tiempo de guardado para la información que tiene cada uno.
> 
> *   **LocalStorage:** Es más persistente que SessionStorage debido a que, como su nombre lo indica, se guarda de manera local en el navegador; así se cierre la pestaña o el dispositivo, la información sigue ahí a menos que elimines el historial o las cookies.
> *   **SessionStorage:** Como su nombre indica, es momentáneo; es memoria temporal, de manera que la información o datos solo serán accesibles mientras la sesión o la pestaña, por ejemplo, estén abiertas.
